### Quick Fixes
1. **Check Logs:** Most issues can be diagnosed from application logs
2. **Restart Services:** Try restarting Docker containers or application services
3. **Clear Cache:** Browser cache can sometimes cause display issues
4. **Check Configuration:** Verify `.env` file settings are correct

## üêõ Report a Bug

If you've found a bug or issue:

1. **Search existing issues:** Your problem may already be reported
2. **Gather information:** 
   - What were you doing when the problem occurred?
   - What error messages did you see?
   - What's your setup? (OS, browser, Docker version, etc.)
3. **Create an issue:** [Open a new issue](https://github.com/afairgiant/MediKeep/issues/new) with:
   - Clear description of the problem
   - Steps to reproduce
   - Expected vs actual behavior
   - Relevant logs or screenshots
